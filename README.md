# 🍳 YUMMY App

An Android application built with **Jetpack Compose** that allows users to discover recipes using data from TheMealDB API. The app features a modern UI inspired by a Figma community design and follows a modular architecture with reusable design components.

## ✨ Features

- Meal Pick of the day
- Different food categories
- Recipes for each meal

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Retrofit
- Kotlin Coroutines
- Coil
- MVVM Architecture

## 📡 API

This application uses **TheMealDB** public API.
The project uses the free development API key (`1`), as specified in TheMealDB documentation. No personal API key is required for the free version.

https://www.themealdb.com/documentation

## 🎨 UI Design

The interface was inspired by the following Figma community design:

https://www.figma.com/design/1wfi12XeCMiHbd38oTaFLv/Grocery-Mobile-App-UI-Design---Prototype--Community-?node-id=48-89&t=ZgOlz6mjcEbSOXO1-0

The project also includes a separate **Design System** module containing reusable Compose UI components.
The project supports both light/dark modes.

## 🚀 Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the application on an emulator or physical device.

NOTE: This app was created as a midterm project for GeoLab Android App Development course.
